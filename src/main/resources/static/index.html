API de Control de Inventario

Demo en vivo de una API REST en Java 17 + Spring Boot: productos, control de stock con auditoría de cada movimiento y alertas de reposición. El dinero se maneja en centavos (sin errores de redondeo). Pruébala ahora mismo en el navegador.

▶ Probar la API (Swagger UI) Especificación OpenAPI (JSON)

REST limpia

CRUD + entradas/salidas de stock, paginación, búsqueda y códigos HTTP correctos (201, 404, 409).

Datos seguros

Operaciones transaccionales con JPA/SQL y registro de auditoría append-only de cada movimiento.

Probada + CI

Pruebas automatizadas con JUnit 5 y pipeline de integración continua en GitHub Actions.

Pruébala desde la terminal

# Listar productos
curl -s $BASE/api/productos

# Crear un producto
curl -s -X POST $BASE/api/productos \
  -H "Content-Type: application/json" \
  -d '{"sku":"LAP-900","nombre":"Laptop 14\"","precioCentavos":1299900,"stockInicial":5,"stockMinimo":2}'

# Registrar una salida (rechaza si no hay stock: HTTP 409)
curl -s -X POST $BASE/api/productos/1/salida \
  -H "Content-Type: application/json" -d '{"cantidad":3,"motivo":"Venta"}'

# Alertas de reposición
curl -s $BASE/api/productos/bajo-stock
