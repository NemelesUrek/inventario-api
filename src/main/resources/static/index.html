Acerca de esta app

Stockly es un sistema de control de inventario para tu negocio, construido por Nemeles. Funciona sobre una API REST documentada (Java + Spring Boot), lista para conectarse con tu tienda en línea, punto de venta u otros sistemas.

A la medida de tu negocio

Inventario, ventas, reservas, reportes. No una plantilla: un sistema que encaja con tu operación real.

Código serio y probado

Java + Spring Boot, con pruebas automatizadas, CI y la API documentada (Swagger/OpenAPI). Listo para producción.
